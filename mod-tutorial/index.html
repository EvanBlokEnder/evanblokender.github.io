Quest Standalone Modding

For standalone Quest modding, the easiest modern method uses the ModsBeforeFriday (MBF) mod installer (the button above links here). This tool lets you mod Beat Saber directly and install custom songs without complex extra tools.

How It Works

  1. Enable Developer Mode — you must enable Meta Developer Mode using the meta app, go into the meta app select the headset icon at the bottom right select the headset name then press settings then developer mode follow the instructions there then enable the toggle switch
  2. Connect Your Quest — plug it into a PC or a Android device with a USB cable.
  3. Visit the MBF Page — click the “Connect to Quest” and follow prompts.
  4. Mod the Game — MBF will patch Beat Saber so custom songs and supported mods work directly in the headset.

Afterward, you can install custom songs through the "Download Songs" button on the left of the main menu of the game no extra PC software needed once MBF is set up.

Note: Some older mod tools like BMBF are no longer maintained for the newest Quest versions reccomeded for quest 1 users — hence MBF is recommended.

PC Modding with BSManager

On PC VR, use a tool called BSManager to manage versions and install mods easily.

Step-by-Step PC Modding

  1. Download BSManager — get it from the official source (GitHub or community links).
  2. Select Game Version — pick the Beat Saber version you want to mod.
  3. Install Core Mods — in BSManager’s “Mods” tab, install core mods (custom songs, UI mods, etc.).
  4. Add Songs & Maps — Use the "Bettersongsearch" mod to install custom songs easily, or the default one "More Songs".
  5. Launch Modded Beat Saber — BSManager can launch the modded version directly.

BSManager also supports features like OneClick™ install for songs, custom sabers, and more mods from trusted community repositories.

Helpful Tips

  • Always backup your game data before modding — especially on Quest.
  • Check mod compatibility — mods often need versions that match your game build.
  • Join communities like Discord or Reddit for support — modding can change over time.
